# Mi Perfil Profesional - Sitio Web Simple

Un sitio web simple en HTML5 puro para mostrar tu perfil profesional. Sin CSS, sin JavaScript, sin colores ni iconos.

## Secciones Incluidas

1. **index.html** - Página principal con navegación
2. **quien-soy.html** - Información personal y habilidades
3. **experiencia.html** - Historial laboral
4. **portafolio.html** - Proyectos realizados
5. **contactame.html** - Formulario de contacto con validación HTML5

## Archivos del Proyecto

```
MiWebPersonal/
├── index.html           # Página de inicio
├── quien-soy.html       # Sección Quién Soy
├── experiencia.html     # Sección Experiencia Laboral
├── portafolio.html      # Sección Portafolio
├── contactame.html      # Sección Contáctame con formulario
└── README.md            # Este archivo
```

## Validación del Formulario

El formulario en `contactame.html` incluye validación HTML5:

- **Nombre**: Requerido, mínimo 3 caracteres, máximo 100
- **Email**: Validación de correo electrónico
- **Teléfono**: Campo opcional
- **Asunto**: Requerido, mínimo 5 caracteres, máximo 100
- **Mensaje**: Requerido, mínimo 10 caracteres, máximo 500
- **Términos**: Checkbox requerido

## Cómo Usar

1. Abre `index.html` en tu navegador
2. Usa el menú para navegar entre secciones
3. Personaliza el contenido según tus datos
4. Actualiza los enlaces de contacto y redes sociales

## Personalización

Simplemente abre cada archivo `.html` en un editor de texto y modifica:
- Nombres, fechas, descripciones
- Enlaces de proyectos
- Datos de contacto
- Información profesional
